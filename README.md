# ft_infinite

The goal of the project is to numbers with no limit of size and be able to operate on them.

## Comparison operators

* Bigger than       >
* Lower than        <
* Bigger or Equal   >=
* Lower or Equal    <=
* Equal             ==
* Different         !=

## Arithmetic operators

* Addition          +
* Subtraction       -
* Multiplication    *
* Division          /
* Modulo            %
* Exponentiation    ^
