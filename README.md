# ft_infinite

The goal of the project is to code numbers with no limit of size and be able to operate on them.

## Operators

### Comparison operators

Name | Symbol
---- | ------
Bigger than     | >
Lower than      | <
Bigger or Equal | >=
Lower or Equal  | <=
Equal           | ==
Different       | !=

### Arithmetic operators

Name | Symbol
---- | ------
Addition        | +
Subtraction     | -
Multiplication  | *
Division        | /
Modulo          | %
Exponentiation  | **

### Bitwise operators

Name | Symbol
---- | ------
Not             | ~
And             | &
Or              | |
Xor             | ^
Right shift     | >>
Left shift      | <<
Right rotate    | >>
Left rotate     | <<
